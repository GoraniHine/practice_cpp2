에러 원인 1:  
반전 구문(a == 2)의 인덱스 처리 오류  

```
for (int j = b; j < c + 1; j++)
{
    if (Light[b - 1] == 0)
    {
        Light[b - 1] = 1;
        b++;
    }
    else if (Light[b - 1] == 1)
    {
        Light[b - 1] = 0;
        b++;
    }
}
```

for문에는 이미 **루프 변수(j)**가 있는데,  
루프 안에서 또 다른 변수(b)를 증가시키면서 동일한 역할을 하여 2중 제어가 되어버림  

해결 방법 1 단일 제어

```
for (int j = b; j <= c; j++) {
    Light[j - 1] = 1 - Light[j - 1];
}
```

요약:  
“2중 제어(두 변수가 동시에 증가)”는 잠재적 버그의 원인  
“단일 제어 변수”로 루프를 돌리는 게 항상 안전하고 명확한 구조
